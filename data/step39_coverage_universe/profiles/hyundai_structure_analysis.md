## 3. 요약표 vs 상세표 차이점

### 3-1. 요약표 ("가입담보 요약표", pages 2-3)

**컬럼 구조**:
```
가입담보 | 가입금액 | 보험료(원) | 납기/만기
```

**특징**:
- **순차 번호로만 담보 구분** (1., 2., 3., ...):
  ```
  1. 기본계약(상해사망)                                   1천만원    448    20년납100세만기
  2. 기본계약(상해후유장해)                               1천만원    550    20년납100세만기
  9. 암진단Ⅱ(유사암제외)담보                             3천만원  35,256   20년납100세만기
  ```
- **카테고리 컬럼 없음** (기본/선택 구분 없음)
- 담보명 전체 기재 (단축 없음)
- **병합된 셀 없음** (LOTTE, HEUNGKUK와 동일)
- 갱신형 담보 표시:
  - 담보명 끝에 "(갱신형)" 표시
  - 납기/만기에 "전기납10년만기갱신(최대100세)" 기재
  ```
  26. 표적항암약물허가치료(갱신형)담보               1천만원    241   전기납10년만기갱신(최대100세)
  ```
- 보험료 합계 단일 표시:
  ```
  보장보험료 합계  119,394원
  ```
  (갱신/비갱신 분리 없음, LOTTE, HEUNGKUK와 동일)

**역할**:
- "어떤 담보에 가입했는가"의 절대 기준
- Universe Lock의 SSOT (Single Source of Truth)

---

### 3-2. 상세표 ("위험보장 및 보험금 지급내용", pages 4-7)

**컬럼 구조**:
```
담보명 및 보장내용 | 납기/만기 | 가입금액 | 보험료(원)
```

**특징**:
- **순차 번호 동일** (요약표와 1:1 대응):
  ```
  9. 암진단Ⅱ(유사암제외)담보

  보장개시일 이후 '암'('유사암'제외)으로 진단 확정된 경우 가입금액 지급(최초1회한)(단, 최초계약일부터 1년미만에 소액암으로 진단시 상기금액의 50%)
  ※유사암 : 기타피부암, 갑상선암, 제자리암, 경계성종양

  20년납100세만기  3천만원  35,256
  ```

- 보장내용 긴 서술형 문장:
  ```
  14. 뇌졸중진단담보

  뇌졸중으로 진단 확정된 경우 가입금액 지급(최초 1회한)(단, 최초계약일부터 1년미만 상기금액의 50%)

  20년납100세만기  1천만원  6,548
  ```

- 조건이 본문에 포함:
  - 보장개시일 (예: 암 90일 대기 - 본문에 명시)
  - 감액 조건 (예: "1년 미만 50% 감액")
  - 제외 사항 (예: "유사암 제외")
  - 지급 한도 (예: "최초 1회한", "연간 1회한")

- **제외사항 별도 섹션 없음** (본문에 통합):
  - 다른 보험사(SAMSUNG, KB, DB, LOTTE, HEUNGKUK)는 "[ 보험금을 지급하지 않는 사항 ]" 형식으로 별도 섹션 존재
  - HYUNDAI는 본문 내에 통합 (예: "※유사암 : 기타피부암, 갑상선암, 제자리암, 경계성종양")

**역할**:
- "이 담보가 어떻게 작동하는가"의 절대 기준
- 고객 질의 응답 시 필수 참조 문서

## 4. 고객 질의와 요약표의 한계

### 4-1. 예시 질의 1

**고객 질문**: "암 진단비는 90일 대기 기간이 있나요?"

**요약표로는 답변 불가**:
- 요약표에는 "9. 암진단Ⅱ(유사암제외)담보 3천만원 35,256 20년납100세만기"만 기재
- 보장개시일 정보 없음

**상세표로만 답변 가능** (page 5):
```
9. 암진단Ⅱ(유사암제외)담보

보장개시일 이후 '암'('유사암'제외)으로 진단 확정된 경우 가입금액 지급(최초1회한)(단, 최초계약일부터 1년미만에 소액암으로 진단시 상기금액의 50%)
※유사암 : 기타피부암, 갑상선암, 제자리암, 경계성종양
```

👉 **답변**: 네, 암 관련 담보는 보장개시일이 있으며, 계약정보(page 4)에 "암관련 담보의 보장개시일은 피보험자의 나이가 보험연령 15세이상인 경우는 계약일로부터 그날을 포함하여 90일이 지난날의 다음날입니다"라고 명시됨.

### 4-2. 예시 질의 2

**고객 질문**: "뇌졸중 진단비는 1년 미만이면 감액되나요?"

**요약표**: "14. 뇌졸중진단담보 1천만원 6,548 20년납100세만기" (조건 정보 없음)

👉 **답변**: 네, 1년 미만이면 50% 감액됩니다.

### 4-3. 결론

**고객 질의 응답은 상세표 없이는 불가능**

## 5. 요약표와 상세표의 관계

### 5-1. 1:1 대응 여부

**기본 원칙**: 1:1 완전 대응

**검증**:
- ✅ 요약표 37개 담보 = 상세표 37개 담보
- ✅ 담보명 정확히 일치
- ✅ 순차 번호 동일 (1→1, 2→2, ...)

**예외 케이스**: **없음**
- 모든 담보가 1:1 대응

---

### 5-2. 요약표에만 있고 상세표에 없는 담보?

**없음**. 요약표의 모든 담보가 상세표에 존재함.

---

### 5-3. 상세표에만 있고 요약표에 없는 담보?

**없음**. 상세표는 요약표의 완전 확장판임.

## 6. STEP 3.9-1 Universe 추출 핵심 판단

### 6-1. Universe 추출 대상 표

**대상 표**: 요약표 "가입담보 요약표" (pages 2-3)

**근거**:
1. **Coverage Universe Lock 원칙 준수**:
   - 가입설계서에서 "고객이 실제 가입한 담보 목록"의 유일한 출처는 요약표
   - 상세표는 보장 조건만 추가 기재할 뿐, 가입 여부를 결정하지 않음

2. **SSOT (Single Source of Truth)**:
   - 요약표 = "What coverages are in this contract"
   - 상세표 = "How each coverage works"

3. **중복 회피**:
   - 요약표와 상세표를 모두 추출하면 동일 담보가 중복 추출됨
   - Universe는 단일 출처에서만 추출해야 함

4. **HYUNDAI 구조적 장점**:
   - 요약표에 병합 셀 없음 (LOTTE, HEUNGKUK와 동일, SAMSUNG보다 파싱 용이)
   - 순차 번호로 명확히 식별 가능
   - 담보명 전체 기재 (단축 없음)

---

### 6-2. 상세표의 역할

**상세표는 Universe 추출 대상이 아님**.

**역할**:
- Universe 추출 후, 각 담보의 조건·감액·보장개시일을 **보강(Augmentation)** 하는 용도
- 고객 질의 응답 시 RAG retrieval 대상 (compare_axis, policy_axis)

**처리 방법**:
1. STEP 3.9-1: 요약표에서 담보명 + 가입금액 + 보험료 추출 → Universe 테이블
2. STEP 후속: 상세표에서 보장내용 + 조건 추출 → Augmentation 또는 별도 테이블

---
