## Example 5: Out of Universe (Edge Case)

### User Query (BLOCK 0)
```
Q. 치아보험을 삼성화재와 메리츠화재 비교해줘
```

### UI Response (No 3-Block Structure)
```
해당 담보는 비교 대상 가입설계서에 포함되지 않아 비교할 수 없습니다.

비교 가능 여부:
  • 삼성화재: 가입설계서에 없음
  • 메리츠화재: 가입설계서에 없음

안내:
본 시스템은 가입설계서에 포함된 담보만 비교합니다.
치아보험 비교를 원하시면 해당 담보가 포함된 가입설계서를 제공해주세요.
```
